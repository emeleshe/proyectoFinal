Conoce Un Poco Más Sobre El Curso

El presente curso ha sido diseñado con el objetivo de abarcar todos los aspectos esenciales para el desarrollo e integración de uno o varios frameworks. Herramientas como Bootstrap, Material Design y UIKit permitirán al desarrollador formar un marco de estándar en sus proyectos personales y profesionales en el mercado competitivo del perfil Front-End.

PERFIL PROFESIONAL

Al finalizar el curso de Front-End Junior podrás:

  • Aprenderás a crear complejos sitios web desde cero
  • Aprenderás a trabajar con la grid de los frameworks para crear sitios web modernos que antes no podíamos..
  • Aprenderás a trabajar con todos y cada uno de los componentes de los frameworks para crear sliders, galerÍas, formularios y casi cualquier cosa que se te ocurra en un sitio web
  • Aprenderás a trabajar con los frameworks y tus proyectos para combinarlo con tus propios diseños
  • Aprenderás a crear sitios web que se adaptan a cualquier pantalla de dispositivo.

TEMARIO

  • Introducción
  • Grid y elementos
  • Components CSS y formularios (Badge, Card, Collapse, Jumbotron, ListGroup, NavBar, Progress, Tolltips)
  • Components Javascript (Modals, Push Notification, Scrollspy, Popovers)
  • Herramientas que se integran (Font-awesome, Facebook API, Google API)
  • Desarrollo de un panel administrativo
  • Introducción: ¿Qué es Materialize?
  • CSS y components (Badges, Cards, Floating Action Button, Icons, NavBar, Preloader).
  • Components Javascript (Auto Init, Dropdown, Modals, Parallax, Pushpin, Scrollspy, Tabs, Toasts, Waves)
  • Formularios & Mobile
  • Desarrollo de un website.
  • Components CSS & Javascript (Alert, animation, Badge, Breadcrumb, Card, Grid, DropDown, Notification, Progress, spinner, Tabs, switcher, Upload)
  • Configuración de los bloques de cursos
  • Desarrollo de un website.

Dirigido A

Emprendedores que deseen diseñar su propia plataforma web, desarrolladores que cuentan con conocimientos básicos Front-end (HTML+CSS+JS) y público en general, interesado en aprender sobre los frameworks.

Metodología y Certificación

La metodología de enseñanza tiene un enfoque teórico-práctico, en la que se analizan las diferentes situaciones y problemáticas que surgen al implementación de un sistema de facturación electrónica y cómo optimizar el tiempo de solución, a la vez que se refuerza la teoría brindada en clase.
