  • Adaptacion de sitios web a culaquier pantalla de dispositivo.
  • Manejo de Flexbox con bootstrap para creacion de sitios web complejos.
  • Versionar tu propio diseño.
  • Creacion de panel de administracion de sitios web y blogs .

La metodología de enseñanza tiene un enfoque práctico en cada una de las sesiones a lo largo del curso. Para el aprendizaje del alumno será necesaria su participación activa, ya que no solo favorecerá a su aprendizaje sino también a la aplicación en situaciones reales en el tema laboral.
