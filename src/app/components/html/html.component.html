Este curso va destinado para personas que no tienen ningún conocimiento en programación y desean ingresar al mundo de la tecnología. Aprenderás a desarrollar una página web responsive (adaptable a móviles) desde cero ya sea para ti mismo, para tu trabajo o para tu propia empresa.

PERFIL PROFESIONAL

Al finalizar el curso de Desarrollo web podrás:

  • Prototipar un sitio web.
  • Maquetar una web con HTML y CSS, siguiendo las mejores prácticas.
  • Crear animaciones con CSS.
  • Manejar SASS como preprocesador, y BEM como metodología.
  • Versionar tú código con GIT y GitHub.
  • Trabajar con Boostrap y con Material Design.
